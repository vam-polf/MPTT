## 布局策略

### 区域划分

```
┌─────────────────────────────────────┐
│  ANT (J2)    E77 (U1)     TP4056    │
│   SMA         LoRa         U4       │
│                                      │
│                       DW01A + 8205A │
│  WM8960 (U2)        U5    +  U6     │
│    Audio                            │
│                                      │
│  J1 USB-C   ME6211    SWD   Batt    │
│             (U3)      (J4)  (J3)    │
└─────────────────────────────────────┘
```

### 布局原则

1. **RF 优先**: 天线区域远离数字噪声源, SMA 座靠近 E77, 50Ω 走线尽量短
2. **电源分区**: 充电/保护电路远离音频模拟区域
3. **去耦就近**: 每个 IC 电源引脚 2mm 内放置去耦电容
4. **GND 完整性**: 底面大面积 GND 铜皮, 顶面保留 GND 回路

### 关键走线

| 走线 | 宽度 | 层 | 说明 |
|------|------|-----|------|
| ANT (RF) | 计算阻抗 | Top | 50Ω 微带线 |
| BAT+ / USB_5V | 30-40 mil | Top | 大电流电源 |
| VBAT_SPK | 30 mil | Top | 功放供电 |
| 信号线 | 12 mil | Top/Bottom | 通用信号 |
| GND | 铺铜 | Bottom | 完整地平面 |

## 元件面分配

全部元件置于 **Top Layer**:
- 52 个 SMD 元件 (Top)
- 1 个直插元件 J4 (2.54mm 2×3P SWD)
- 0 个元件在 Bottom

## 生产文件

通过 EasyEDA Pro → 文件 → 导出 → 嘉立创PCB下单 直接生成 Gerber。
或在 EDA 中导出:
- Gerber (RS-274X)
- NC Drill
- BOM (CSV)
- CPL 坐标文件 (CSV)

## SMT 贴片

- 只贴顶层
- JLCPCB 标准型 SMT
- E77 和 WM8960 需邮寄物料 (嘉立创无现货)

## 已知限制

- WM8960 已停产，需从淘宝/库存采购
- RF 天线走线未做严格阻抗仿真 (第一版原型)
- 无屏蔽罩，RF 可能干扰音频电路
