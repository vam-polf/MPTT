# 原理图设计说明 v0.1

> EasyEDA Pro | 单页原理图 P1 | 53 个元件 | 作者: vam-polf | 2026-06-07

## 功能模块划分

### 1. 核心控制 — E77-400M22S (U1)

STM32WLE5 内置 LoRa 射频的单片机模组。负责:
- LoRa 无线收发 (400MHz)
- I2S 音频数据流控制
- I2C 配置 WM8960
- GPIO 控制 PTT/充电状态
- ADC 电池电压采样

关键引脚连接:
- PA4/PA5/PA6/PA7 → I2S 到 WM8960
- PB6/PB7 → I2C 到 WM8960
- PB0 → PTT 按键
- PA0 → BAT_ADC
- PA13/PA14 → SWD (J4)

### 2. 音频编解码 — WM8960 (U2)

立体声音频 Codec + 1W D类功放。

模拟部分:
- MIC+/LINPUT1 → 麦克风输入
- SPK_LP/SPK_LN → 扬声器输出 (BTL 桥接)
- MICBIAS → 驻极体偏置

数字部分:
- I2S 4线接口 (MCLK/BCLK/LRCLK/SD)
- I2C 配置接口 (SCL/SDA)

电源分组:
- +3.3V_DVDD: 数字核心
- +3.3V_BVDD: 缓冲器
- +3.3V_AVDD: 模拟电路
- VBAT_SPK: 功放级 (直连电池)

### 3. 电源管理

```
USB-C 5V → TP4056 → DW01A → 8205A → BAT+
                         ↓
                    ME6211 → +3.3V → 各路供电
```

- **TP4056 (U4)**: 恒流/恒压充电, 200mA 充电电流, LED 状态指示
- **DW01A (U5)**: 过充(4.25V)/过放(2.5V)/过流保护
- **8205A (U6)**: 双 MOSFET 开关, 切断电池回路
- **ME6211 (U3)**: 3.3V 500mA LDO, 输入6V, 低压差

### 4. USB-C 接口 (J1)

- VBUS → USB_5V (充电输入)
- D+/D- → E77 USB (预留, 当前未连)
- CC1/CC2 → 各接 5.1kΩ 下拉 (R17/R18)

### 5. 天线接口 (J2)

SMA 母座, 50Ω 特性阻抗。PCB 走线需控制阻抗。

### 6. 外设接口

| 连接器 | 用途 |
|--------|------|
| J3 (1.25mm 2P) | 锂电池连接 |
| J4 (2.54mm 2×3P) | SWD 调试 (SWDIO/SWCLK/NRST/GND) |
| J5 (ZH 1.5mm 2P) | 扬声器 8Ω/1W |
| J6 (ZH 1.5mm 2P) | 驻极体麦克风 |

## 设计规则

- 原理图坐标单位: 0.01inch (10 mil)
- 元件总数: 53
- 网络总数: 40
- DRC 状态: 通过

## 去耦电容布局

每个 IC 电源引脚就近放置 100nF + 10μF/22μF 去耦电容:
- E77: C9/C10/C11/C12/C13/C14/C15/C16/C17 + 电源隔离 (R7/R8/R9/R10/R11 0Ω 跳线)
- WM8960: C1/C2/C3/C4/C5 + 各电源域独立去耦
- ME6211: 输入 C18 (1μF), 输出大电容组
- TP4056: 输入/输出电容
