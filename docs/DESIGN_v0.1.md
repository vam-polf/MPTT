## 1. 系统概述

MPTT v2.5 是一款基于 STM32WLE5 的 LoRa 手持对讲机。单节锂电池供电，USB-C 充电，SMA 外接天线。

### 1.1 核心能力

| 功能 | 实现 |
|------|------|
| 无线通信 | E77-400M22S LoRa 扩频模块, 400-470MHz |
| 音频采集 | 驻极体麦克风 → WM8960 ADC → I2S → MCU |
| 音频播放 | MCU → I2S → WM8960 DAC → 1W D类功放 → 扬声器 |
| 电源 | 单节锂电池, TP4056 充电, DW01A 保护, ME6211 LDO |
| 调试 | SWD 2×3P 接口 |
| 充电 | USB-C (5V/200mA) |

### 1.2 设计选型理由

选用**模组方案**（E77-400M22S）而非裸芯片，原因：
- 避免 RF 阻抗匹配和天线设计的复杂性
- 模组已过 FCC/CE 认证
- 内置 STM32WLE5 + RF 前端，缩小 PCB 面积
- 简单、稳定、可靠

选用**WM8960 模拟 Codec** 而非数字 MEMS 麦克风方案，原因：
- 驻极体麦克风成本极低（~1元），数字 MEMS 需 5-10 元
- WM8960 内置 1W D 类功放，无需额外功放 IC
- 立体声能力为未来扩展预留

- VCC 由 `+3.3V_E77` 供电（通过 0Ω 跳线 R7/R8/R9/R10/R11 与主 3.3V 隔离）
- 多组去耦电容: C9(100nF), C10(100nF), C11(10μF), C12(100nF), C13(100nF), C14(22μF), C15(100nF), C16(10μF), C17(10μF)
- 0Ω 跳线允许独立测量 E77 功耗

## 4. 音频子系统: WM8960

### 4.1 Codec 规格

| 参数 | 值 |
|------|-----|
| 型号 | WM8960CGEFL/RV |
| 封装 | QFN-32 (5×5mm) |
| DAC SNR | 98dB |
| ADC SNR | 94dB |
| 功放输出 | 1W @ 8Ω (D类, BTL) |
| 耳机输出 | 40mW @ 16Ω |
| 麦克风 | 差分/单端, 内置 PGA + ALC |
| 接口 | I2S (标准/左对齐/DSP), I2C 控制 |
| 状态 | ⚠️ 已停产, 需淘宝/库存采购 |

### 4.2 信号链

```
TX路径 (发送):
  Electret Mic → MIC+/LINPUT1 → WM8960 PGA → ADC → I2S_SD → E77 → LoRa TX

RX路径 (接收):
  LoRa RX → E77 → I2S_SD → WM8960 DAC → D类功放 → SPK_LP/SPK_LN → Speaker (J5)
```

### 4.3 WM8960 引脚连接

#### I2S 数字音频

| WM8960 引脚 | 信号 | 连接 |
|-----------|------|------|
| 13 | ADCLRC | E77 PA4 (I2S1_WS) |
| 14 | DACLRC | E77 PA7 (I2S1_MCK) — 复用为 LRC |
| 15 | BCLK | E77 PA5 (I2S1_CK) |
| 16 | ADCDAT | E77 PA6 (I2S1_SD) |
| 17 | DACDAT | E77 PA7 — 双向数据 |

#### I2C 控制

| WM8960 引脚 | 连接 |
|-----------|------|
| 19 (SCLK) | E77 PB6 (I2C1_SCL) |
| 20 (SDIN) | E77 PB7 (I2C1_SDA) |

I2C 地址: 0x1A (7-bit, CSB=GND)

#### 模拟音频

| WM8960 引脚 | 功能 | 外部连接 |
|-----------|------|---------|
| 8 (LINPUT1) | 麦克风输入 | 驻极体 MIC+ (J6), 隔直电容 |
| 9 (MICBIAS) | 麦克风偏置 | 通过电阻供电给驻极体 |
| 25 (SPK_LP) | 扬声器 正 | J5-1 (ZH1.5 2P) |
| 26 (SPK_LN) | 扬声器 负 | J5-2 |

SPK_LP/SPK_LN 为 BTL 桥接输出，无需隔直电容。

#### 电源分组

WM8960 有多组独立电源域，需分别去耦：

| 电源引脚 | 网络 | 去耦 |
|---------|------|------|
| 1, 2, 31 (AVDD) | +3.3V_AVDD | C1(22μF) + C2(100nF) + C3(100nF) |
| 10, 18 (DVDD) | +3.3V_DVDD | C5(10μF) + C6(100nF) |
| 22 (SPKVDD1) | VBAT_SPK | 电池直供, C7(22μF) + C8(100nF) |
| 28 (SPKVDD2) | VBAT_SPK | 同上 |
| 12 (DBVDD) | +3.3V_BVDD | 缓冲供电 |

**设计要点**: SPKVDD 直接接电池电压 (3.7-4.2V)，不经过 3.3V LDO。D 类功放在较高电压下效率更好。

### 4.4 麦克风电路

- 驻极体麦克风 (6027 单指向, -42dB, 6×2.7mm)
- MICBIAS 提供偏置电压 (~2.0V via WM8960 内部)
- 隔直电容 (C4) 耦合到 LINPUT1
- 差分输入可抑制共模噪声

## 5. 电源管理

### 5.1 电源架构

```
USB-C 5V (J1)
   │
   ▼
┌──────────┐    ┌──────────┐    ┌──────────┐
│  TP4056  │───►│  DW01A   │───►│  8205A   │──► BAT+ (J3)
│  Charger │    │ Protect  │    │ MOSFET   │
│  U4      │    │  U5      │    │  U6      │
└────┬─────┘    └──────────┘    └──────────┘
     │
     ├── CHRG ──► U8 (Red LED)  充电指示
     └── STDBY ─► U9 (Green LED) 充满指示

BAT+ ──► ME6211 (U3) ──► +3.3V ──┬── +3.3V_E77 (经0Ω跳线)
                                  ├── +3.3V_AVDD
                                  ├── +3.3V_DVDD
                                  └── +3.3V_BVDD

BAT+ ──► VBAT_SPK (WM8960 功放直供)
```

### 5.2 TP4056 充电管理 (U4)

| 参数 | 值 | 设置方式 |
|------|-----|---------|
| 充电电流 | ~200mA | R17//R18 → 5.1kΩ → PROG 引脚 |
| 终止电流 | C/10 (~20mA) | 内部固定 |
| 充电电压 | 4.2V ±1% | 内部固定 |
| 输入电压 | 4.5-6V (USB-C 5V) | |
| 温度保护 | 禁用 | TEMP 引脚接 GND |

充电状态指示:
- CHRG (开漏): 充电中 → LED U8 (红) 亮
- STDBY (开漏): 充满 → LED U9 (绿) 亮
- 两个 LED 均通过限流电阻 (R16: U8, R14: U9) 连接到 +3.3V

**充电电流计算**: I_CHG = 1000V / R_PROG
- R17 = 5.1kΩ, R18 = 5.1kΩ 并联 → R_PROG ≈ 2.55kΩ
- I_CHG ≈ 392mA (理论值)
- 实际限流约 200mA (考虑 USB 供电能力)

### 5.3 DW01A 电池保护 (U5) + 8205A MOSFET (U6)

DW01A 监控电池状态，通过 8205A 双 MOSFET 切断充放电回路：

| 保护功能 | 阈值 | 动作 |
|---------|------|------|
| 过充保护 | 4.25V ±50mV | OC→高, 切断充电 MOSFET |
| 过放保护 | 2.40V ±100mV | OD→高, 切断放电 MOSFET |
| 放电过流 | 150mV (≈3A) | OD→高 |
| 短路保护 | 1.5V | 立即切断 |

8205A: 双 N 沟道 MOSFET, Vds=20V, Id=6A, Rds(on)=25mΩ

### 5.4 ME6211 LDO (U3)

| 参数 | 值 |
|------|-----|
| 型号 | ME6211C33M5G-N |
| 封装 | SOT-23-5 |
| 输入电压 | 最大 6V |
| 输出电压 | 3.3V 固定 |
| 输出电流 | 最大 500mA |
| 压差 | 100mV @ 200mA |
| LCSC | C82942 |

输入电容 C18 (1μF), 输出电容由各路去耦组成。

### 5.5 功耗估算

| 工作模式 | E77 | WM8960 | 其他 | 总计 |
|---------|-----|--------|------|------|
| RX 监听 | ~8mA | ~2mA | ~1mA | ~11mA |
| TX (22dBm) | ~120mA | ~10mA | ~1mA | ~131mA |
| 待机 | ~2μA | 0 (掉电) | ~5μA | ~7μA |

500mAh 电池续航: RX ~45小时, TX ~3.5小时

## 6. 接口与外设

### 6.1 USB-C (J1)

| 引脚 | 连接 | 说明 |
|------|------|------|
| VBUS | USB_5V | 充电输入 |
| D+/D- | NC | 数据线未使用 (E77 USB 预留) |
| CC1 | 5.1kΩ 下拉 (R17) | 配置为 UFP (受电设备) |
| CC2 | 5.1kΩ 下拉 (R18) | 同上 |
| GND | GND | |

Type-C 母座, 卧贴封装。CC1/CC2 各接 5.1kΩ 到 GND → USB-C 主机识别为需要 5V 供电的设备。

### 6.2 SMA 天线 (J2)

- 型号: SMA-KE-5 (母座, 正脚)
- 50Ω 特性阻抗
- 走线尽量短, 参考地层完整
- 无额外匹配网络 (E77 模组内部已匹配)

### 6.3 SWD 调试接口 (J4)

2×3P, 2.54mm 间距, 直插方针:

| 引脚 | 信号 |
|------|------|
| 1 | +3.3V |
| 2 | SWDIO |
| 3 | GND |
| 4 | SWCLK |
| 5 | GND |
| 6 | NRST |

### 6.4 电池连接器 (J3)

1×2P, 1.25mm PicoBlade 卧贴:
- Pin 1: BAT+ (正极)
- Pin 2: GND (负极)

### 6.5 扬声器 (J5)

ZH 1.5mm 1×2P 立贴:
- Pin 1: SPK_LP (WM8960 输出正)
- Pin 2: SPK_LN (WM8960 输出负)

推荐扬声器: 8Ω/1W, 直径 20mm 腔体

### 6.6 麦克风 (J6)

ZH 1.5mm 1×2P 立贴:
- Pin 1: MIC+ (经隔直电容到 WM8960 LINPUT1)
- Pin 2: GND

推荐麦克风: 6027 单指向驻极体, -42dB

### 6.7 PTT 按键 (SW1)

- 型号: TS-1187A-B-A-B (5.1×5.1×1.5mm 立贴)
- 连接: PB0 → SW1 → GND
- 内部上拉: E77 PB0 内置弱上拉
- 外部下拉: R12 (4.7kΩ) 确保默认低电平

### 6.8 电池电压检测

- PA0 (ADC) ← R19 (100kΩ) ← BAT+ ← R20 (100kΩ) → GND
- 分压比: 1/2
- 满电 4.2V → ADC 读 2.1V

## 7. 电源树与去耦策略

### 7.1 全局去耦布局

```
+3.3V ──┬── C18(1μF)   ← ME6211 输出
        ├── C5(10μF)    ← WM8960 DVDD
        ├── C1(22μF)    ← WM8960 AVDD
        ├── C2,C3,C6,C8,C9,C10,C12,C13,C15 (100nF×9)
        └── C11,C16,C17 (10μF×3)

VBAT_SPK ──┬── C7(22μF)
           └── C14(22μF)  ← WM8960 SPKVDD

+3.3V_E77 ── C11,C14,C16,C17  ← 经0Ω隔离后

BAT+ ── C4(22μF)  ← TP4056 输入
```

### 7.2 电源域隔离

| 隔离方式 | 网络 | 目的 |
|---------|------|------|
| 0Ω 跳线 | +3.3V → +3.3V_E77 | 可断开独立测量 E77 电流 |
| 独立 LDO | +3.3V_AVDD | 模数隔离 (暂用同一 LDO) |
| 直连电池 | VBAT_SPK | D类功放高压高效 |


### 9.3 关键走线规则

| 信号 | 线宽 | 间距 | 层 | 说明 |
|------|------|------|-----|------|
| RF ANT | 计算 50Ω | ≥20mil 到 GND | Top | 微带线, 尽量短 |
| VBAT_SPK | 30mil | — | Top | 功放大电流 |
| BAT+ | 30mil | — | Top | 主供电 |
| USB_5V | 30mil | — | Top | 充电输入 |
| I2S 信号 | 12mil | ≥12mil | Top | 等长 (±5mil) |
| I2C | 12mil | — | Top/Bottom | 非高速 |
| GND | 铺铜 | — | Bottom | 完整地平面 |

### 9.4 注意事项

1. **RF 隔离**: 天线区域远离数字/电源噪声源，地平面完整无割裂
2. **音频隔离**: WM8960 模拟区域 (AVDD/SPKVDD) 与数字区域之间用 0Ω 跳线隔离地回路
3. **SWD 防护**: NRST 线上拉 100kΩ，SWDIO/SWCLK 串 100Ω 保护
4. **散热**: TP4056 在 200mA 充电时功耗约 0.3W，无需额外散热
5. **去耦就近**: 每个 IC 电源引脚 2mm 内放置 100nF 电容

## 10. 设计决策与取舍

### 10.1 模组 vs 裸芯片

| 方案 | 优点 | 缺点 |
|------|------|------|
| E77 模组 (✅ 选用) | 免 RF 匹配/认证, 邮票孔易焊接, 尺寸小 | 成本较高 (~30元), 引脚固定 |
| 裸 STM32WLE5 + RF | 灵活, 成本低 | RF 设计复杂, 需阻抗匹配, 需认证 |

### 10.2 WM8960 模拟 Codec vs 数字 MEMS 方案

| 方案 | 优点 | 缺点 |
|------|------|------|
| WM8960 (✅ 选用) | 内置功放, 麦克风便宜, 立体声 | 芯片停产, QFN 焊接难度 |
| INMP441 + MAX98357A | 全数字, 噪声低, 芯片在产 | 成本高 (~15元), 需要独立功放, 更多 PCB 面积 |

### 10.3 USB-C 充电 vs 专用充电口

选用 USB-C 母座 (UFP 模式)，兼容主流充电器和数据线。CC1/CC2 各 5.1kΩ 下拉 → 标准 5V 供电。

### 10.4 已知限制

1. **WM8960 停产**: 需淘宝/库存采购，长期维护需考虑替换方案
2. **无屏蔽罩**: RF 可能干扰音频电路，需实测评估
3. **RF 天线未做阻抗仿真**: 第一版原型基于模组厂家参考设计
4. **2 层 PCB**: 无专门的地平面层，需底层大面积铺铜
5. **无 ESD 保护**: USB/SMA/各连接器未加 TVS，后续版本可增加
